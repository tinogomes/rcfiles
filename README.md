# My RC Files

## Requirements

Oh my zsh - https://github.com/tinogomes/oh-my-zsh

    $ git clone git://github.com/tinogomes/oh-my-zsh.git .oh-my-zsh

## How to install

    $ cd $HOME
    $ git clone git://github.com/tinogomes/rcfiles.git .rcfiles

## Local settings

Write into ~/.zshrc_local file

    $ touch ~/.zshrc_local
