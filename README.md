# My RC Files

## Requirements

Oh my zsh - https://github.com/tinogomes/oh-my-zsh

    $ git clone git://github.com/tinogomes/oh-my-zsh.git

## How to install

    $ cd $HOME
    $ git clone git://github.com/tinogomes/rcfiles.git .rcfiles
    $ .rcfiles/install

## Local settings

Create .zshrc\_local file at your ```$HOME```

    $ touch ~/.zshrc_local
